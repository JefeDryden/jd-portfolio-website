Shopper Behaviour Prediction Model

Machine learning model to predict online shopping conversion rates

School Project Python, Machine Learning, Predictive Analytics

Project Overview

This project developed a machine learning model to predict whether online shopping sessions would result in purchases. By analyzing user behavior patterns, session characteristics, and temporal factors, the model helps e-commerce businesses understand and predict customer conversion likelihood.

The project demonstrates the application of data science in e-commerce optimization, providing actionable insights for improving conversion rates and customer experience.

Data Analysis & Features

Behavioral Metrics

Analyzed page views, session duration, bounce rates, and exit rates to understand user engagement patterns.

Temporal Features

Incorporated time-based features including day of week, month, special shopping periods, and seasonal trends.

User Characteristics

Examined visitor type (new vs. returning), browser, operating system, and traffic source impacts.

Page Categories

Analyzed user interaction with different page types: product, administrative, and informational pages.

Machine Learning Pipeline

  • Data Preprocessing: Handled missing values, encoded categorical variables, and normalized numerical features
  • Feature Engineering: Created interaction features and derived metrics from raw session data
  • Model Selection: Tested multiple algorithms including Random Forest, XGBoost, and Logistic Regression
  • Hyperparameter Tuning: Used grid search and cross-validation for optimal model configuration
  • Evaluation: Assessed model performance using accuracy, precision, recall, and F1-score

Key Insights

The analysis revealed several important factors influencing purchase likelihood:

  • Page value emerged as the strongest predictor of conversion
  • Session duration and page views showed non-linear relationships with conversion
  • Temporal factors, especially proximity to special shopping events, significantly impacted conversion rates
  • Returning visitors demonstrated higher conversion rates than new visitors
  • Mobile users showed different behavior patterns compared to desktop users

Model Performance

The final model achieved strong predictive performance, enabling accurate identification of high-conversion-probability sessions. This allows for:

  • Real-time personalization of user experience
  • Targeted intervention for sessions at risk of abandonment
  • Optimization of marketing spend on high-value visitors
  • Better understanding of factors driving purchase decisions
