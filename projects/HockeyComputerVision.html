NHL Computer Vision Tracking

Using YOLOv4 to detect and track hockey players in live game footage

Personal Project Python, YOLOv4, Computer Vision View on GitHub

Project Overview

This project combines my passion for data analytics, hockey, and personal family history. Using computer vision techniques, I developed a system to detect, identify, and track hockey players, goalies, and referees in live game TV footage.

What makes this project unique is its personal connection: the initial model was trained on footage from my grandfather's first NHL game as an emergency backup goalie on February 3rd, 1962. This blend of cutting-edge technology with family history created a meaningful exploration of computer vision applied to hockey analytics.

Technical Approach

YOLOv4 Object Detection

Implemented YOLOv4 (You Only Look Once) for real-time object detection, specifically trained to identify hockey players, goalies, and referees.

Position Mapping

Developed an innovative approach using ice advertisements as anchor points to map player positions on the rink.

Training Data Generation

Created a tool to generate new training data from model outputs, enabling continuous improvement of the detection system.

Key Features

  • Real-time detection and tracking of players, goalies, and referees
  • Ability to process both historical (1962) and modern NHL footage
  • Custom training data generation tool for model improvement
  • Position mapping using ice surface reference points
  • Potential to create new, meaningful hockey statistics from broadcast data
